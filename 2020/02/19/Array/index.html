Array(数组)

1.特点：

a.数组一旦被定义，大小不可改变（java特性，长度一开始就定义好了）

b.数组是引用数据类型，但是数组可以被定义成存储任意类型8个基本类型

 c.数组中的数据类型必须是一致的，和定义的类型一致数组在创建时就一定要声明类型，然后数组就只能存储这个类            型

d.数组有默认值，默认是就是定义类型的默认值

     byte short int ---0

     long--0L
     float double ---0.0

     char---\u000

     String----null

e.数组有顺序，以下标定义 从0-length-1
